1. 	vertical-align: ;		=> 	căn lề
	margin-top: ;			=>	căn lề trên với tp khác

2. CSS Grid <Bố cục lưới>:	
---Code:
	display: grid
	grid-template-columns: 100px 1fr 	lưới (1,2) size: 100px và free(lấy hết không gian còn lại)
	column-gap: 20px			(khoảng cách giữa các cột)
	row-gap: 10px 				(khoảng cách giữa các hàng)
3. Flexbox
---Code:
	display: flex;
        flex-direction: row;		(trình bày giữa dạng hàng)
        flex: 1;			(=1fr/grid)
	justify-content: center;	(căn các cột trong box)
	align-items: stretch;		(lấp đầy không gian theo chiều dọc)
	max-width: 400px;		(max size của box)
	flex-shrink: 0;			(d_(2 elements)=const)
   => tạo 1 box 
		
		Note: 	flex-shrink: 0; = no shrink
			width = 0;	= shrink
